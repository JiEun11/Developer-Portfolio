## 🚀 Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Set environment variables

Create a `.env.local` file:

```bash
GEMINI_API_KEY=your_api_key
```

### 3. Run the app

```bash
npm run dev
```
